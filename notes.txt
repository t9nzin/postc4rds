4 main HTTP methods (actions)

1. POST - create new resource
---
// Create a new postcard
POST /api/postcard

2. GET - read/retrieve resource 
---
// Get a specific postcard
GET /api/postcard/[id]

// Get all postcards
GET /api/postcards

3. PATCH (or PUT) - update existing resource 
// Update postcard message
PATCH /api/postcard/[id]

4. DELETE - remove resource 
// Delete a postcard
DELETE /api/postcard/[id]
